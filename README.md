## Description

[Nest](https://github.com/nestjs/nest) Framework TypeScript starter repository.<p>
[TypeORM](https://typeorm.io/#/) TypeORM is an ORM.<p>
[Swagger/OpenAPI](https://swagger.io/) OpenAPI Specification & Swagger Tools.<p>

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Support

[typeorm-model-generator](https://github.com/Kononnable/typeorm-model-generator) Generates models for TypeORM from existing databases
```bash
# typeorm-model-generator -h <host> -d <database> -p [port] -u <user> -x [password] -e [engine]
$ typeorm-model-generator -h localhost -d "secret" -p "secret" -u postgres -x "secret" -e postgres
```

Migrations
Create migrations

```bash

# create ormconfig file
# obs: Ao gerar o arquivo ormconfig.json verificar se os paths estão com dist/ 
# ao criar um migration remover 
$ npm run pretypeormconfig

# generate migrations
$ npm run typeorm:migration:generate

# run migrations
$ npm run typeorm:migration:run

# create model entities (optional)
$ typeorm entity:create -d src/entities -n

# CLI query table (optional)
$ npm run query -- "select * from customers"
```
